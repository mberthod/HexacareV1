Detailed Description

Driver UART HLK-LD6002 — protocole TinyFrame, ring buffer UART.

Protocole TinyFrame : [SOF=0x01][Type_H][Type_L][Longueur][Données...][EOF=0x04]

Types reconnus : 0x0A14 : Fréquence respiratoire (2 octets big-endian, bpm) 0x0A15 : Fréquence cardiaque (2 octets big-endian, bpm) 0x0A16 : Distance de la cible (2 octets big-endian, mm)

Le driver UART (UART_NUM_2) est déjà installé par hw_diag_run(). radar_driver_init() se contente de réinitialiser l'état du parser. radar_driver_poll() est appelé à chaque cycle de Task_Sensor_Acq.
