Detailed Description

Driver Texas Instruments HDC1080 — Température et Humidité relative.

Interface I2C via driver/i2c_master.h (ESP-IDF v5+). Adresse I2C 7 bits : 0x40 (fixe sur HDC1080).

Caractéristiques :

  • Température : −40 °C → +125 °C, résolution 14 bits (±0.2 °C typ.)
  • Humidité : 0 % → 100 % RH, résolution 14 bits (±2 % typ.)
  • Temps d'acquisition : ~15 ms (mode 14 bits)
