Detailed Description

Interface publique du gestionnaire de réseau maillé ESP-NOW.

Topologie arborescente (Tree Mesh) :

  • Chaque nœud maintient une liste de pairs (peers).
  • Sélection du parent : peer avec le RSSI le plus élevé parmi ceux ayant un hop_to_root minimal.
  • Chiffrement : PMK/LMK ESP-NOW (couche hardware) + AES-128-CBC sur le payload applicatif (couche logicielle, mbedTLS).

Clé AES-128 : Provisionnée en NVS (namespace "lexacare", clé "aes_key") lors de la fabrication. Si absente, une clé par défaut est utilisée (NON recommandé en production).
