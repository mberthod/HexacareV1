Detailed Description

Adaptateur plateforme pour l'ULD ST VL53L8CX (STSW-IMG040).

L'ULD ST inclut un fichier platform.h que le client doit fournir. Dans ce firmware, la couche plateforme (SPI) est implémentée par : components/sensor_acq/include/vl53l8cx_platform.h

Ce fichier fait le lien entre l'ULD (API ST) et l'implémentation ESP-IDF.
