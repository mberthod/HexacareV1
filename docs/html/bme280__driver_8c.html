Detailed Description

Driver Bosch BME280 — Température, Pression, Humidité.

Implémentation de l'algorithme de compensation entier 32/64 bits issu du datasheet Bosch BME280 (v1.24, Appendix A).

Mode forced (single-shot) : déclenche une mesure puis passe en sleep. Oversampling T×2, P×4, H×4 — filtre IIR × 4.
