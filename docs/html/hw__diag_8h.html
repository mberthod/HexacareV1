Detailed Description

Interface publique du diagnostic matériel au démarrage.

hw_diag_run() doit être appelée dans app_main() AVANT la création des tâches FreeRTOS. Elle initialise les bus I2C partagés et remplit les champs lidar_ok[] et radar_ok de sys_context_t.
