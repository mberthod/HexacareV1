Detailed Description

Interface publique du driver Radar HLK-LD6002 — UART + TinyFrame.

Le driver utilise le ring buffer UART installé par hw_diag_run() (uart_driver_install sur UART_NUM_2). Il parse le protocole TinyFrame : SOF(0x01) | Type(2o) | Longueur(1o) | Données | EOF(0x04)

Types reconnus : 0x0A14 → breath_rate_bpm (uint16 big-endian) 0x0A15 → heart_rate_bpm (uint16 big-endian) 0x0A16 → target_distance_mm (uint16 big-endian)
