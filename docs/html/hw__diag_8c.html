Detailed Description

Diagnostic matériel au démarrage — I2C LIDAR + UART Radar.

Exécuté avant les tâches FreeRTOS dans app_main(). Initialise les bus I2C partagés et les stocke dans sys_context_t pour réutilisation par sensor_acq sans réinitialisation.
