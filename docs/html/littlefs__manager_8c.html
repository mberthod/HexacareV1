Detailed Description

Gestion du montage SPIFFS sur la partition "storage".

Implémentation via esp_spiffs (built-in ESP-IDF v5.4). L'API publique reste identique — les fichiers POSIX (fopen/fclose/etc.) fonctionnent de la même façon avec SPIFFS qu'avec LittleFS.

Note : SPIFFS ne supporte pas les répertoires. Tous les fichiers sont à la racine du point de montage (/spiffs/).
