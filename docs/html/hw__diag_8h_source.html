40/* ================================================================
41 * hw_diag_run
42 * @brief Exécute le diagnostic matériel complet et remplit sys_context_t.
43 *
44 * Séquence :
45 * 1. Initialise I2C_NUM_0 (SDA=8, SCL=9) et I2C_NUM_1 (SDA=16, SCL=15).
46 * 2. Met tous les LPn à 0 (reset global de tous les VL53L8CX).
47 * 3. Active les LIDARs un par un : LPn=1, attente 5 ms,
48 * probe I2C à LIDAR_I2C_ADDR_DEFAULT (0x29),
49 * si OK → réassigne l'adresse à (LIDAR_I2C_ADDR_BASE + i).
50 * 4. Configure UART2 à 1382400 bauds (LD6002).
51 * Écoute RX pendant 2000 ms et valide la trame TinyFrame.
52 * 5. Formate le rapport JSON avec cJSON et l'imprime sur la console.
53 *
54 * @param ctx Pointeur vers le contexte système (rempli par cette fonction).
55 * @return Masque hw_diag_result_t (0 = succès total).
56 * ================================================================ */
57hw_diag_result_t hw_diag_run(sys_context_t *ctx);
