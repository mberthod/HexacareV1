Detailed Description

Point d'entrée : initialise les services, lance les diagnostics, puis démarre les tâches.

L'intention est de garder une séquence de boot lisible et “déterministe” : chaque étape prépare la suivante, et évite de lancer des tâches si une dépendance manque.
