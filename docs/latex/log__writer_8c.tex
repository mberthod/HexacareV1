\doxysubsection{Detailed Description}
Log circulaire sur Little\+FS — rotation automatique à 900 Ko. 

Algorithme de rotation \+:
\begin{DoxyEnumerate}
\item Ouvrir le fichier en mode "{}a"{} (append).
\item Écrire le message horodaté.
\item Si ftell() dépasse LOG\+\_\+\+MAX\+\_\+\+SIZE\+\_\+\+BYTES \+: a. Lire tout le fichier en mémoire (allocation heap). b. Calculer le nombre d\textquotesingle{}octets à ignorer (LOG\+\_\+\+TRIM\+\_\+\+PERCENT \%). c. Avancer jusqu\textquotesingle{}à la fin de la ligne correspondante. d. Réécrire le reste du fichier depuis le début.
\end{DoxyEnumerate}

Thread-\/safety \+: mutex Free\+RTOS créé à la première écriture.
