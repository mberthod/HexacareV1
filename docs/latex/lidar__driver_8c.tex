\doxysubsection{Detailed Description}
Acquisition LIDAR VL53\+L8\+CX — fusion 8×32 sur double bus I2C. 

Architecture dual-\/bus (contrainte débit) \+: I2\+C\+\_\+\+NUM\+\_\+0 → capteurs 0 \& 1 (lus en parallèle avec bus 1) I2\+C\+\_\+\+NUM\+\_\+1 → capteurs 2, 3 \& 4

Fusion \+: capteur N → colonnes \mbox{[}N\texorpdfstring{$\ast$}{*}8 .. N\texorpdfstring{$\ast$}{*}8+7\mbox{]} de \doxylink{structlidar__matrix__t}{lidar\+\_\+matrix\+\_\+t}. Cadence \+: 15 Hz via v\+Task\+Delay\+Until (période 66 ms). TWDT \+: esp\+\_\+task\+\_\+wdt\+\_\+reset() à chaque itération.

Dépendance ULD \+: Les appels vl53l8cx\+\_\+\texorpdfstring{$\ast$}{*}() nécessitent l\textquotesingle{}ULD VL53\+L8\+CX de ST. Voir instructions dans \doxylink{vl53l8cx__platform_8h}{vl53l8cx\+\_\+platform.\+h}. Sans l\textquotesingle{}ULD, compiler avec -\/DLIDAR\+\_\+\+STUB\+\_\+\+MODE pour utiliser des données simulées (développement sans matériel).
