\doxysubsection{Detailed Description}
Adaptateur plateforme pour l\textquotesingle{}ULD ST VL53\+L8\+CX (STSW-\/\+IMG040). 

L\textquotesingle{}ULD ST inclut un fichier {\ttfamily \doxylink{platform_8h}{platform.\+h}} que le client doit fournir. Dans ce firmware, la couche plateforme (SPI) est implémentée par \+: \doxylink{vl53l8cx__platform_8h}{components/sensor\+\_\+acq/include/vl53l8cx\+\_\+platform.\+h}
