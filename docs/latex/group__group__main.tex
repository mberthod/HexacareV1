\doxysubsection{Detailed Description}
Point d\textquotesingle{}entrée \+: initialise les services, lance les diagnostics, puis démarre les tâches. 

L\textquotesingle{}intention est de garder une séquence de boot lisible et “déterministe” \+: chaque étape prépare la suivante, et évite de lancer des tâches si une dépendance manque.
