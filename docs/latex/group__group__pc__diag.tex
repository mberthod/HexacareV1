\doxysubsection{Detailed Description}
Sortie USB en JSON lisible + commandes simples depuis l\textquotesingle{}outil PC. 

Ce module sert de “fenêtre” sur le firmware \+:
\begin{DoxyItemize}
\item envoie un état régulier (santé système + mesures capteurs)
\item permet quelques actions (récupérer/effacer les logs) sans recompiler 
\end{DoxyItemize}
