\doxysubsection{Detailed Description}

Topologie arborescente (Tree Mesh) \+:
\begin{DoxyItemize}

\item Chiffrement \+: PMK/\+LMK ESP-\/\+NOW (couche hardware) + AES-\/128-\/\+CBC sur le payload applicatif (couche logicielle, mbed\+TLS).
