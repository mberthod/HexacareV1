\doxysubsection{Detailed Description}
Driver Texas Instruments HDC1080 — Température et Humidité relative. 

Interface I2C via driver/i2c\+\_\+master.\+h (ESP-\/\+IDF v5+). Adresse I2C 7 bits \+: 0x40 (fixe sur HDC1080).

Caractéristiques \+:
\begin{DoxyItemize}
\item Température \+: −40 °C → +125 °C, résolution 14 bits (±0.2 °C typ.)
\item Humidité \+: 0 \% → 100 \% RH, résolution 14 bits (±2 \% typ.)
\item Temps d\textquotesingle{}acquisition \+: \texorpdfstring{$\sim$}{\string~}15 ms (mode 14 bits) 
\end{DoxyItemize}
