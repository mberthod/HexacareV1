\doxysubsection{Detailed Description}
Structure \doxylink{struct_v_l53_l8_c_x___results_data}{VL53\+L8\+CX\+\_\+\+Results\+Data} contains the ranging results of VL53\+L8\+CX. If user wants more than 1 target per zone, the results can be split into 2 sub-\/groups \+: 


\begin{DoxyItemize}
\item Per zone results. These results are common to all targets (ambient\+\_\+per\+\_\+spad , nb\+\_\+target\+\_\+detected and nb\+\_\+spads\+\_\+enabled).
\item Per target results \+: These results are different relative to the detected target (signal\+\_\+per\+\_\+spad, range\+\_\+sigma\+\_\+mm, distance\+\_\+mm, reflectance, target\+\_\+status). 
\end{DoxyItemize}
