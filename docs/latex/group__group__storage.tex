\doxysubsection{Detailed Description}
Gestion du stockage local (SPIFFS) et mécanismes de log/panic. 

Pourquoi ce module existe \+:
\begin{DoxyItemize}
\item conserver des traces (logs) même si le PC n\textquotesingle{}est pas connecté
\item aider au diagnostic après un redémarrage inattendu (panic handler) 
\end{DoxyItemize}
