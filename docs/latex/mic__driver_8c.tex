\doxysubsection{Detailed Description}
Driver microphone MEMS I2S — Amplitude RMS et crête. 

Utilise le driver I2S standard d\textquotesingle{}ESP-\/\+IDF v5/v6 (driver/i2s\+\_\+std.\+h). Le microphone MEMS (ex \+: INMP441, SPH0645) émet des données en I2S 32 bits, canal gauche (ou droit selon LRCK). Les 24 bits supérieurs contiennent les données audio, les 8 bits inférieurs sont nuls.
