\doxysubsection{Detailed Description}
Lecture capteurs, fusion des données, et production de trames prêtes pour l\textquotesingle{}IA. 

Ce module existe pour “isoler le monde matériel” \+:
\begin{DoxyItemize}
\item la partie IA reçoit une trame stable et cohérente (mêmes unités, mêmes dimensions)
\item la partie diagnostic PC reçoit des mesures prêtes à afficher 
\end{DoxyItemize}
