# ESP-IDF v6.0 — Documentation (Markdown)
Généré à partir des sources Sphinx officielles (ESP-IDF v6.0, mars 2026).
**Usage recommandé** : pointer un RAG (Khoj, Claude Code, Cursor) sur ce dossier. Les fichiers gardent la hiérarchie d'origine pour faciliter la recherche sémantique.

**Note sur les placeholders** : les textes comme `{IDF_TARGET_NAME}` ne sont pas résolus (Sphinx les remplace au build selon la cible ESP32/ESP32-S3/etc.). Préciser la cible à l'IA au besoin.
