Open the ESP-IDF Installation Manager application `eim`.

Under `Manage Installations`, click `Open Dashboard`.

<figure>
<img src="../../_static/get-started-eim-gui.png" class="align-center" alt="../../_static/get-started-eim-gui.png" />
<figcaption>EIM Manage Installations</figcaption>
</figure>

In the dashboard, you will see all the installed ESP-IDF versions. Select the version you want to use, and click `Open IDF Terminal` to launch a terminal session with activated ESP-IDF environment.

<figure>
<img src="../../_static/get-started-eim-gui-open-terminal.png" class="align-center" alt="../../_static/get-started-eim-gui-open-terminal.png" />
<figcaption>EIM Open IDF Terminal</figcaption>
</figure>
