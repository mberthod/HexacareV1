# ESP-Modbus

The Espressif ESP-Modbus Library (esp-modbus) supports Modbus communication in the networks based on RS485, Wi-Fi, and Ethernet interfaces. The component `esp-modbus v2 (v2.x.x)` is the current supported component version:

- [ESP-Modbus component on GitHub](https://github.com/espressif/esp-modbus/tree/main)
- [ESP-Modbus component in component manager](https://components.espressif.com/components/espressif/esp-modbus)

## Hosted Documentation

The documentation can be found through the link below:

- [ESP-Modbus V2 documentation (English)](https://docs.espressif.com/projects/esp-modbus/en/stable)
